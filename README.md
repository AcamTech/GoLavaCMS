# Install
- Download or clone.
- Change your database, username, password on .env file
- ```php artisan key:generate```
- ```php artisan migrate```
- Register new member
- Done


# About
GoLavaCMS is based on the [Laravel 5.5](https://github.com/laravel/laravel). Frontend on the Bootstrap framework created by  [Mark Otto](https://twitter.com/mdo) and [Jacob Thorton](https://twitter.com/fat) & [Bootswatch](https://github.com/thomaspark/bootswatch). Theme By [Startbootstrap](https://github.com/blackrockdigital).
